xapi-jqm

Integration of xAPI with jQuery Mobile

View project onGitHub

What?

Integration of the Experience API (xAPI) using jQuery Mobile (jQM) version 1.4.2 with demos and templates based on xapi-jqm which handle the heavy lifting in communicating statements to a Learning Record Store (LRS).

Instructional designers and developers can focus on content and HTML rather than writing javascript code, without limiting their abilities to extend their statements and tracking of interactions.

Check out visual reports generated from statements.

Who?

Intended for teachers, instructional designers, developers and more!
